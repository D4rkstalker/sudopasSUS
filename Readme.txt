==================================================================
Controls

Directional Keys
'W' Key / UP Arrow Key - Move UP
'A' Key / LEFT Arrow Key - Move LEFT
'S' Key / Down Arrow Key - Move DOWN
'D' Key / RIGHT Arrow Key - Move RIGHT

Skill/Trigger Keys
Space Key - To trigger off the checkpoint
Left Mouse Button (LMB/MB1) - Focused Ping
Right Mouse Button (RMB/MB2) - Area of effect Ping

Control Keys (Can only be used after control part of the tutorial)
'M' Key - Map
Escape Key - Pause Menu  

==================================================================
Menu features

- Resume Game
- Volume (Slider)
- Retry from last checkpoint
- Retry Game 
- Exit Game
